My Projects

Tourism Management System

I developed a menu-driven application in C to streamline the management of tourist details, tour packages, and booking processes. This system was designed to enhance the operational functionality and improve the user experience by utilizing advanced data structures such as arrays and linked lists. By implementing these structures, I ensured efficient organization and management of records, ultimately optimizing the application's performance. Through this project, I strengthened my programming skills in C, developed robust algorithms, and improved my ability to handle data reliably and efficiently.

Sentiment Analysis of Metaverse Tweets

In this project, I applied machine learning techniques to analyze the sentiment of tweets related to the Metaverse. Using Python, I processed and refined text data through powerful NLP tools like NLTK and TextBlob, achieving an impressive accuracy rate of 85%. I enhanced the model's performance through feature engineering and hyperparameter tuning, ensuring high reliability in sentiment classification. This project sharpened my skills in natural language processing and reinforced my understanding of text-based data analysis, while demonstrating my ability to extract valuable insights from complex data sets.

Lung Cancer Prediction

I created a predictive model for lung cancer detection using Python and various powerful libraries such as NumPy, Pandas, Scikit-Learn, OpenCV, and Matplotlib. The project involved data acquisition, preprocessing, exploratory data analysis (EDA), and feature selection, with the model achieving an accuracy rate of 87%. Through rigorous model evaluation and fine-tuning, I improved the precision of the predictions. This project not only allowed me to apply machine learning algorithms to a real-world healthcare challenge but also enhanced my analytical proficiency and deepened my understanding of data science techniques in the medical domain.

Music Recommendation System

For the Music Recommendation System project, I developed a unique recommendation engine using the Spotify API and machine learning techniques. I utilized a dataset containing detailed audio features, track metadata, and user preferences to build a personalized recommendation system. By employing content-based filtering, I recommended songs based on similarities in attributes such as genre, tempo, and mood. Additionally, I integrated machine learning models, including Principal Component Analysis (PCA) for dimensionality reduction and DBSCAN for clustering, along with cosine similarity to refine recommendations. The system offered an intuitive interface that provided users with personalized music suggestions, enhancing their listening experience with data-driven recommendations.
